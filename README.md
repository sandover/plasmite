# plasmite

CLI and library for plasmite.

## Install

```bash
cargo install plasmite
```

### Homebrew

Create your tap repo and copy the formula from `homebrew/plasmite.rb`.
See `docs/homebrew.md` for the full steps.

```bash
brew tap YOUR_GITHUB/tap
brew install plasmite
```

## Usage

```bash
plasmite
```

See `docs/exit-codes.md` for the stable exit code mapping.

## Development

```bash
cargo test
```
